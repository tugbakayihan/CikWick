CikWick Game Project

CikWick is a game developed by following the tutorial from this video
, which walks through creating interactive game mechanics, such as movement and object interaction, within Unity.

Features:
Player Movement: Navigate the environment with intuitive controls.
Interaction System: The game includes interactive elements like checking water levels.
Smooth Animations: The character performs actions with responsive and smooth animations.

Acknowledgments:

This project was developed by following an educational tutorial on YouTube. The tutorial provides step-by-step instructions on how to create the game mechanics.
